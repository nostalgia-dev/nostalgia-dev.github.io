Nostalgia at the highest level consists of 3 components. Sources are the connections coming from the data into the system. The Core is faciliting the interface that sources should adhere to. It also provides an immediate query language (based on Python pandas) that provides helper functions to working with and combining this data. Then lastly, there are Apps that can utilize the data to do things, like for example displaying it using a Timeline or querying it using a Chatbot.
